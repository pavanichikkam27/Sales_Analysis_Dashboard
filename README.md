# Sales_Analysis_Dashboard
Interactive Sales Analysis Dashboard built using Microsoft Excel. The project analyzes sales data by region, product, and month using Pivot Tables, Charts, Slicers, Timeline, and KPI cards. It helps visualize trends, compare performance, and generate business insights through an easy-to-use dashboard
Tools Used:
- Pivot Tables
- Pivot Charts
- Slicers
- Timeline
- KPI Card
Insights:
- Total Sales: 410300
- Total Orders: 20
- Average Sales: 20515
